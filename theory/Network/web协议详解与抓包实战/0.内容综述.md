## HTTP 1.0的协议升级

​	HTTP 1.0 协议是无状态的，不支持服务器主动向客户端推送消息，有些页面需要实时刷新消息时，没有办法做到，轮询可以实现，但是效率低下。

-   支持服务器推送消息的WebSocket协议
    -   建立会话
    -   消息传输
    -   心跳关闭会话
-   全面优化后的HTTP 2.0协议（websocket在多路复用上表现一般）
-   HTTP 2.0必须开启TLS/SSL协议
